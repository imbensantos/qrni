# Auth Flow Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:executing-plans to implement this plan task-by-task.

**Goal:** Add a Google OAuth sign-in modal overlay and header profile dropdown so users can sign in/out without leaving the app.

**Architecture:** AuthModal component renders at the App level, controlled by state lifted into App.jsx. An `onRequestAuth` callback is threaded down to child components that need it. When triggered, the modal opens; after Google sign-in completes, the modal auto-closes. The header shows a profile dropdown with sign-out when authenticated.

**Tech Stack:** React 19, Convex Auth (`@convex-dev/auth/react`), CSS (no extra libraries)

### Task 1: Create AuthModal component

**Files:**

- Create: `apps/app/src/components/AuthModal.jsx`

**Step 1: Create AuthModal.jsx**

```jsx
import { useAuthActions } from "@convex-dev/auth/react";
import "./AuthModal.css";

function AuthModal({ open, onClose, message }) {
  const { signIn } = useAuthActions();

  if (!open) return null;

  const handleSignIn = async () => {
    try {
      await signIn("google");
    } catch (err) {
      console.error("Sign-in error:", err);
    }
  };

  return (
    <div className="auth-backdrop" onClick={onClose}>
      <div
        className="auth-modal"
        onClick={(e) => e.stopPropagation()}
        role="dialog"
        aria-modal="true"
        aria-labelledby="auth-title"
      >
        <button className="auth-close" onClick={onClose} aria-label="Close">
          <svg width="16" height="16" viewBox="0 0 16 16" fill="none">
            <path
              d="M12 4L4 12M4 4l8 8"
              stroke="currentColor"
              strokeWidth="1.5"
              strokeLinecap="round"
            />
          </svg>
        </button>
        <h2 id="auth-title" className="auth-title">
          QRni ✨
        </h2>
        <p className="auth-subtitle">
          {message ||
            "Sign in to unlock custom slugs, namespaces, and link history."}
        </p>
        <button className="auth-google-btn" onClick={handleSignIn}>
          <svg width="18" height="18" viewBox="0 0 18 18" aria-hidden="true">
            <path
              d="M17.64 9.2c0-.637-.057-1.251-.164-1.84H9v3.481h4.844a4.14 4.14 0 01-1.796 2.716v2.259h2.908c1.702-1.567 2.684-3.875 2.684-6.615z"
              fill="#4285F4"
            />
            <path
              d="M9 18c2.43 0 4.467-.806 5.956-2.18l-2.908-2.259c-.806.54-1.837.86-3.048.86-2.344 0-4.328-1.584-5.036-3.711H.957v2.332A8.997 8.997 0 009 18z"
              fill="#34A853"
            />
            <path
              d="M3.964 10.71A5.41 5.41 0 013.682 9c0-.593.102-1.17.282-1.71V4.958H.957A8.996 8.996 0 000 9c0 1.452.348 2.827.957 4.042l3.007-2.332z"
              fill="#FBBC05"
            />
            <path
              d="M9 3.58c1.321 0 2.508.454 3.44 1.345l2.582-2.58C13.463.891 11.426 0 9 0A8.997 8.997 0 00.957 4.958L3.964 6.29C4.672 4.163 6.656 2.58 9 2.58z"
              fill="#EA4335"
            />
          </svg>
          Continue with Google
        </button>
      </div>
    </div>
  );
}

export default AuthModal;
```
